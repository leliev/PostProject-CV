Présentation

Bienvenu sur mon premier projet de CV version Web

Ce site représente un simple exercice et d'autres projets plus complexes sont à venir.
Cette version utilise du HTML 5, CSS 3 et un peu de javaScript pour la gestion des slides/articles et celle du thème de couleur.
L'idée de ce site est de ressembler le plus possible à un CV conventionnel et correspond mieux à un format A4 (avec une approche "mobile first") mais sa simplicité le rend relativement bien responsive.
Vous trouverez mes info personnelles sur la version pdf de mon CV (cette version étant publique).
